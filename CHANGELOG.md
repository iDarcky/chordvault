## [Unreleased]

### Added
- Core architecture setup (Vite, React, Tailwind v4, HeroUI v3).
- Comprehensive `.md` parser handling chords, sections, band cues, modulations, and tabs.
- Local storage implementation using `idb-keyval`.
- Three-mode Editor: Visual (with toolbar), Form, and Raw.
- Setlist Builder with reordering and per-song transposition.
- Live performance modes (auto-responsive layouts, large text).
- Bluetooth pedal support for navigation and scrolling.
- Capo calculator and shape rendering.
- Import/Export functionality for individual `.md` files and `.zip` setlists.
- Cloud Sync engine infrastructure with adapters for Google Drive, Dropbox, and OneDrive.
- Product documentation, including How-To Guide, Privacy Policy, and Architecture docs.
